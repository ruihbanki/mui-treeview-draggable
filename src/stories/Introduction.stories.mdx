# Introduction

The TreeView and TreeItem are wraps of the [Material UI](https://material-ui.com/components) components with adicional props to support drag and drop operations.
The DataTreeView is a component that takes a tree data json to create a tree. The drag and drop operation and its props were inspired by the [antd](https://ant.design/components/tree/) tree component.

## Goals

- Provide drag and drop support
- Work on desktop and mobile devices
- Provide support to screens readers (a11y)

## Accessibility

These components have support to accessibility providing keyboard only control and use of aria attributes.

## Install

yarn add mui-treeview-draggable

## How to use

```
import { TreeView, TreeItem, DataTreeView } from "mui-treeview-draggable";
```
