# Introduction

The TreeView component is a wrap of the Material UI [TreeView](https://material-ui.com/components/tree-view/#tree-view) with adicional props to support drag and drop.

## Goals

- Provide drag and drop support
- Work on desktop and mobile devices
- Provide support to screens readers (a11y)

## Accessibility

A drag and drop operation can be done on a selected node, keeping the space bar pressed.
The up and down arrow keys allow users to navigate to the desired drop location and confirm with the Enter key.
